## Requirements
- requirements.txt
- python 3.8
- virtualenv 20.0.17
- Flask 2.2.3

## Overview
- Thu thập dữ liệu
- Gán nhãn dữ liệu
- Kiểm thử thuật toán naive-bayes
- Phân loại trên dữ liệu mới

## Install
```bash
git clone https://github.com/dt140120/Flask_twitter.git
cd Flask_twitter
pip3 install . -r requirements.txt
```

## Run
```bash
cd Flask_twitter
python3 app.py
```
